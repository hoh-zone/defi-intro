# 第 9 章 CDP 与稳定币

## 什么是 CDP

CDP（Collateralized Debt Position）是"抵押 + 铸造"的机制：用户抵押某种资产，协议铸造出稳定币。用户偿还稳定币后，取回抵押品。

CDP 本质上是一个自托管的稳定币印刷机。但"稳定"不是自然属性——它需要精心的参数设计、预言机集成和清算机制来维持。
