# 第 5 章 预言机：链上价格事实

## 什么是预言机

预言机（Oracle）是把链下数据搬上链的桥梁。在 DeFi 中，预言机最核心的用途是提供**价格数据**。

但"价格"不是一个简单的概念。当你看到一个数字时，你需要问：这个数字从哪来？谁在维护它？它多久更新一次？如果它错了会怎样？

## 本章的核心论点

预言机不是一个"数据接口"。它是一组**信任假设**的集合。当你选择使用某个预言机时，你不是在选一个数据源，而是在选一组你愿意接受的信任边界。
