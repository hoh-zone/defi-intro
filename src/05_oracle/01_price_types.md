# 5.1 市场价格 ≠ 协议价格 ≠ 价格事实

## 三种价格

### 市场价格（Market Price）

市场上的成交价格。由供需决定，实时变化。DEX 的最后一笔成交价就是一种市场价格。

特征：高频率、有噪声、反映即时供需。

### 协议价格（Protocol Price）

协议内部使用的价格。通常是市场价格的某种处理版本——时间加权平均（TWAP）、多源中位数、带置信区间等。

特征：滞后于市场、过滤了噪声、但可能错过真实的价格跳变。

### 价格事实（Price Fact）

经过验证的价格声明。它不仅包含价格数值，还包含：

- 数据来源
- 更新时间戳
- 置信区间
- 发布者签名

```move
public struct PriceFact has copy, drop, store {
    price: u64,
    conf: u64,
    publish_time: u64,
    publisher: address,
    feed_id: ID,
}
```

特征：可验证、带元数据、信任边界明确。

## 为什么区分很重要

| 场景           | 用市场价格 | 用协议价格 | 用价格事实 |
| -------------- | ---------- | ---------- | ---------- |
| DEX swap 定价  | 是         | 否         | 否         |
| 借贷抵押率计算 | 否         | 否         | 是         |
| 清算触发       | 否         | 否         | 是         |
| 展示用行情     | 是         | 是         | 否         |

借贷和清算是风险最高的操作——如果价格错了，用户要么被意外清算，要么协议产生坏账。这些场景必须使用价格事实，而不是原始市场价格。

## 不同协议对价格的依赖方式

### 借贷协议

需要"当前价格"来计算：

- 存款价值（决定能借多少）
- 借款价值（决定抵押率）
- 是否触发清算

对价格的敏感度：**极高**。一次错误的价格更新可能导致大量不当清算。

### CDP 稳定币

需要"稳定价格"来维持锚定。CDP 的铸造和赎回都依赖价格。如果价格被操纵，攻击者可以用少量抵押品铸造大量稳定币。

对价格的敏感度：**极高且系统性**——稳定币脱锚会影响整个生态。

### 衍生品

需要"连续价格"来计算保证金、资金费率和强平。衍生品对价格的频率和精度要求最高。

对价格的敏感度：**极高**——标记价格直接决定是否强平。
