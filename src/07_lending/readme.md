# 第 6 章 借贷：从储蓄池到信用市场

## 为什么从借贷开始

借贷是 DeFi 中第一个真正复杂的协议类型。它比 DEX 多了利率模型、抵押品管理、清算机制和预言机依赖。理解借贷，就为理解 CDP、衍生品等更复杂的协议打好了基础。

## 本章的三个层次

| 层次 | 协议 | 复杂度 | 核心机制 |
|------|------|--------|----------|
| 储蓄池 | Sui Savings | 低 | 存入、取出、领息 |
| 借贷市场 | Lending Protocol | 中 | 存借、利率模型、抵押清算 |
| 闪电贷 | Flash Loan | 中高 | 无抵押瞬时借贷 |

每一步都配完整的 Move 代码实现。代码不是伪代码——是可以在 Sui 测试网上部署的真实合约结构。
