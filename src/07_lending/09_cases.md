# 7.9 借贷全景对比

## 五种借贷范式

```
储蓄池 → 资金池借贷 → 闪电贷 → 隔离市场 → 零利率市场
(最简单)  (行业标准)   (工具)    (风险隔离)  (特定场景)
```

## 全景对比表

| 维度 | 储蓄池 | 资金池借贷 | 闪电贷 | 隔离市场 | 零利率市场 |
|------|--------|-----------|--------|----------|-----------|
| 借款人 | 无 | 有 | 有（瞬时） | 有 | 有 |
| 抵押品 | 无 | 多种共享 | 无 | 每对独立 | 多种共享 |
| 利率 | 固定 | 动态拐点 | 一次性费用 | 动态拐点 | 零 |
| 清算 | 无 | 有 | 不适用 | 有 | 有 |
| 风险范围 | 最小 | 跨资产 | 交易级 | 交易对级 | 跨资产 |
| 代表 | Sui Savings | Aave, Compound | Aave 闪电贷 | Euler, Silo | Scallop |

## 协议对比

| 协议 | 类型 | 链 | 核心创新 | 适合场景 |
|------|------|-----|----------|----------|
| Aave V4 | 资金池 | 多链 | 统一流动性层 | 通用借贷 |
| Compound V4 | 资金池→模块化 | 多链 | 模块化架构 | 可组合借贷基础设施 |
| Navi | 资金池 | Sui | 自动杠杆 | Sui 上的通用借贷 |
| Scallop | 资金池+零利率 | Sui | Vault + 零利率 | 稳定币借贷 |
| Euler | 隔离市场 | 多链 | 反应性利率 | 长尾资产借贷 |
| Silo | 隔离市场 | 多链 | 桥接资产 | 简化隔离市场 |

## 利率模型对比

| 模型 | 使用者 | 特点 |
|------|--------|------|
| 固定利率 | 早期协议 | 简单但不灵活 |
| 线性利率 | 早期 DAI | 有响应但保护不足 |
| 拐点模型 | Aave, Compound, Navi, Scallop | 行业标准 |
| 动态利率 | Aave V3+ | 渐进调整，减少跳跃 |
| 反应性利率 | Euler | 根据利用率和时间自动调参 |

## 清算机制对比

| 机制 | 使用者 | 描述 |
|------|--------|------|
| 固定折扣清算 | Aave V2 | 清算者获得固定折扣购买抵押品 |
| Dutch Auction | Aave V3 | 折扣从低到高，时间越长折扣越大 |
| Absorb | Compound V3 | 协议直接吸收坏账，用储备金覆盖 |
| 部分清算 | Navi, Scallop | 只清算部分债务，保留用户仓位 |

## 选择框架

```
你要做什么？
│
├── 理解借贷基础 → 储蓄池 (7.1)
│
├── 通用借贷需求
│   ├── 在 Sui 上 → Navi (7.6) 或 Scallop (7.7)
│   └── 在 EVM 上 → Aave V3 (7.4) 或 Compound V3 (7.5)
│
├── 瞬时无抵押借款 → 闪电贷 (7.3)
│
├── 长尾资产借贷 → 隔离市场 (7.8)
│
├── 稳定币零成本借贷 → Scallop 零利率市场 (7.7)
│
└── 构建新借贷协议
    ├── 参考架构 → Compound V4 模块化 (7.5)
    └── 风险隔离 → Euler 隔离市场 (7.8)
```

## 借贷协议的安全底线

无论哪种类型，所有借贷协议都需要：

| 检查项 | 说明 |
|--------|------|
| 预言机 | 多源 + TWAP + 偏差校验 |
| 利率模型 | 极端利用率下的行为已验证 |
| 清算激励 | 确保清算者有利润 |
| 暂停机制 | 紧急暂停功能 |
| 存借款上限 | 防止集中风险 |
| 坏账处理 | 储备金或 absorb 机制 |
| 治理权限 | 多签 + 时间锁 |
