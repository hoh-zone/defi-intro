# 第 13 章 现货杠杆：从借贷到放大的交易

## 现货杠杆 vs 永续合约

永续合约是"持有价格敞口而不持有资产"。现货杠杆是**持有更多资产**——通过借贷放大购买力。

区别：

- 永续合约：你持有的是合约，不是资产
- 现货杠杆：你持有的是真实资产，只是部分资金是借来的

在 Sui 上，现货杠杆的实现依赖借贷协议（如 Cetus 的借贷模块）和 DEX 的组合。

## 本章小结

现货杠杆不是新的资产类型，而是把借贷和交易连接起来的路径。Sui 的 PTB 让这种组合更自然，也要求协议在原子性、滑点、债务上限和清算顺序上更谨慎。
