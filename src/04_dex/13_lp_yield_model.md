# 4.13 LP 收益模型

LP 的总收益由两部分组成：手续费收入和无常损失。本节建立 LP 收益的完整分析模型。

## LP 总收益公式

```
LP 总收益 = 手续费收入 - 无常损失 + 价格变动收益

或者用年化收益率表示:
  LP APY = Fee APY - |IL| + Price Change

注意：LP 也承受代币价格变动的收益/损失（这不是 IL，而是正常的持仓损益）
```

## Fee APY 计算

### 基本公式

```
Fee APY = (日手续费收入 / TVL) × 365

日手续费收入 = 日交易量 × 费率 × LP 分配比例

例：
  日交易量: $500,000
  费率: 0.3%
  LP 分配: 80%
  TVL: $1,000,000

  日手续费 = $500,000 × 0.3% × 80% = $1,200
  Fee APY = ($1,200 / $1,000,000) × 365 = 43.8%
```

### 影响因素

| 因素 | 对 Fee APY 的影响 |
|------|-----------------|
| 交易量增加 | Fee APY 上升 |
| TVL 增加 | Fee APY 下降（同样的交易量稀释到更大 TVL） |
| 费率提高 | Fee APY 上升（但交易量可能下降） |
| LP 分配比例增加 | Fee APY 上升 |

### Fee APY 的不稳定性

```
Fee APY 随市场条件波动极大:

牛市高峰:
  交易量 = $2M/天, TVL = $1M
  Fee APY = ($2M × 0.3% × 80% / $1M) × 365 = 175%

熊市低谷:
  交易量 = $100K/天, TVL = $1M
  Fee APY = ($100K × 0.3% × 80% / $1M) × 365 = 8.8%

差异: 20 倍
```

## APR vs APY

```
APR（年化百分比利率）:
  不考虑复利
  APR = Fee APY（简单计算）

APY（年化百分比收益）:
  考虑复利
  APY = (1 + APR/n)^n - 1
  n = 复利次数

在 AMM 中，手续费自动留在池中 = 自动复利:
  如果 Fee APR = 50%
  复利频率 = 每笔交易（假设日均 100 笔）
  APY ≈ (1 + 50%/36500)^36500 - 1 ≈ 64.9%

实际中，AMM 的 Fee APR 和 Fee APY 差异不大（因为不是每次交易都复投）
```

## 净收益分析

### 盈亏平衡分析

```
LP 盈亏平衡条件:
  Fee Income ≥ |IL|

Fee Income = TVL × Fee APY × time
|IL| = TVL × IL_rate

盈亏平衡的 Fee APY:
  Fee APY ≥ IL_rate / time

例：
  价格变化 2x（IL = 5.72%）
  LP 持有时间 = 30 天
  盈亏平衡 Fee APY = 5.72% / (30/365) = 69.6%

  如果实际 Fee APY < 69.6% → LP 在 30 天内亏损
  如果实际 Fee APY > 69.6% → LP 在 30 天内盈利
```

### 不同交易对的收益特征

| 交易对类型 | Fee APY（典型）| IL（典型）| 净收益 | 评价 |
|-----------|--------------|----------|--------|------|
| 稳定币-稳定币 | 5-15% | ~0% | 5-15% | 最稳定 |
| 主流币-主流币 | 20-50% | 2-5% | 15-45% | 较好 |
| 主流币-稳定币 | 30-80% | 5-15% | 15-65% | 波动大 |
| 山寨币-主流币 | 50-200% | 10-40% | -10% ~ +160% | 高风险 |

## 波动性对 LP 收益的影响

```
价格波动性对 LP 收益的双重影响:

正面: 高波动 → 更多套利交易 → 更多手续费收入
负面: 高波动 → 更大 IL → 更多无常损失

净效果取决于哪个因素主导

经验法则:
  低波动 (稳定币): 手续费主导 → LP 稳定盈利
  中波动 (主流币): 不确定 → 需要具体分析
  高波动 (山寨币): IL 可能主导 → LP 可能亏损
```

## 收益归因

```
LP 总收益拆解:
                        ┌─ 价格收益（持仓本身的涨跌）
LP 总 PnL ─── 市场收益 ─┤
              │         └─ IL（AMM 机制导致的损失）
              │
              └─ 手续费收入 ─── LP Fee ─── 直接留在池中
                              └─ Protocol Fee → 协议金库

例：
  LP 投入 $4,000（2000 SUI + 2000 USDC，SUI = $2）
  30 天后 SUI = $2.50

  市场收益:
    HODL 价值 = 2000 × $2.50 + $2000 = $7,000
    初始投入 = $4,000
    价格收益 = +$3,000 (+75%)

  IL:
    AMM 价值 ≈ $6,750（比 HODL 少 $250）
    IL = -$250 (-3.6%)

  手续费收入:
    Fee APY = 40%, 30 天
    手续费 = $4,000 × 40% × 30/365 = $131

  LP 总 PnL = +$3,000 - $250 + $131 = +$2,881 (+72%)
```

## Sui 上 LP 的额外考量

Sui 的特性对 LP 收益有独特影响：

1. **低 Gas 费** → LP 再平衡成本更低（调整 CLMM 区间）
2. **PTB 原子操作** → 可以在一个交易中收取手续费 + 再投资
3. **并行执行** → 高频交易不受其他交易对影响
4. **对象模型** → LP Position 可以作为其他协议的抵押品（增加收益来源）
