# 4.23 StableSwap 曲线

稳定币之间的兑换（如 USDC ↔ USDT）对价格极度敏感——0.1% 的偏差在稳定币世界就是大问题。CPMM 的滑点对稳定币来说太大了，需要专门的曲线。

## CPMM 为什么不适合稳定币

```
USDC/USDT 池（CPMM）:
  reserve_a (USDC) = 1,000,000
  reserve_b (USDT) = 1,000,000

交易 $100,000 USDC → USDT（费率 0.3%）:
  amount_out = 99,700 × 1,000,000 / 1,099,700 = 90,646
  价格冲击 = 9.35%
  实际损失 ≈ $9,354

这对稳定币交易者来说完全不可接受！
稳定币应该 1:1 兑换，损失不应该超过 0.01%
```

## StableSwap 不变量

Curve 提出的 StableSwap 不变量是恒定和（x+y=k）和恒定积（x·y=k）之间的插值：

```
恒定和: x + y = k  → 价格永远 1:1，但池会被掏空
恒定积: x × y = k  → 价格灵活，但滑点大

StableSwap: 在两者之间平滑过渡

公式（简化两币版）:
  A × (x + y) + x × y = D²/4 + A × D

其中:
  D = 池的"不变量"（类似 CPMM 中的 k）
  A = 放大系数（amplification coefficient）
```

### 放大系数 A 的作用

```
A 很大（如 A=1000）:
  → 曲线接近 x+y=k（恒定和）
  → 在平衡点附近，滑点极小
  → 适合稳定币（价格应该接近 1:1）

A 很小（如 A=0.01）:
  → 曲线接近 x×y=k（恒定积）
  → 滑点较大
  → 更像普通 AMM

A = 0:
  → 退化为标准 CPMM
```

### 曲线形状对比

```
价格
  │         CPMM (x·y=k)
  │        /
  │       /
  │      /    StableSwap (A=100)
  │     /    ─────────────────
  │    /    ╱
  │   /    ╱  （在平衡点附近几乎平直）
  │  /    ╱
  │ /    ╱
  │/____╱───────────────── x+y=k（恒定和）
  └──────────────────────── 储备比例

在平衡点附近，StableSwap 曲线几乎与 x+y=k 重合
→ 滑点极小
远离平衡点时，曲线弯向 x·y=k
→ 防止池被掏空
```

## 滑点对比

```
USDC/USDT 池（各有 1,000,000），交易 $100,000:

CPMM:
  滑点 ≈ 9.1%（完全不可接受）

StableSwap (A=100):
  滑点 ≈ 0.02%（几乎无损）

StableSwap (A=10):
  滑点 ≈ 0.15%

稳定币标准:
  可接受滑点 < 0.05%
  → A ≥ 50 基本满足要求
```

## StableSwap 的数学细节

### D 的计算

```
给定池中各代币的储备量 x_1, x_2, ..., x_n，求 D:

A × Σ(x_i) + Π(x_i) = D^n/(n^n) + A × D

这是一个关于 D 的非线性方程，需要用 Newton 法迭代求解:

D_{k+1} = D_k - f(D_k) / f'(D_k)

其中:
  f(D) = A × n^n × Σ(x_i) + n^n × Π(x_i) - A × n^n × D - D^n
```

### Swap 输出计算

```
已知输入 Δx，求输出 Δy:

1. 已知池的当前状态，计算 D
2. 令 x_new = x + Δx
3. 从 StableSwap 方程中解出 y_new
4. Δy = y - y_new

同样需要 Newton 法迭代
```

## 动态 A 参数

A 不是固定不变的，可以根据市场条件动态调整：

```
正常市场: A = 100 → 极低滑点
轻度脱锚: A = 50 → 中等滑点，但保护池
严重脱锚: A = 10 → 接近 CPMM，保护池不被掏空

调整逻辑:
  当 |price - 1| < 0.5%: A = 100
  当 0.5% < |price - 1| < 2%: A = 50
  当 |price - 1| > 2%: A = 10

目的: 在平衡时提供低滑点，在极端时保护 LP
```

## 脱锚场景分析

```
USDC 脱锚（价格跌至 $0.90）:

CPMM 池:
  大量 USDC 被换入，USDT 被换出
  LP 损失 = USDT 流出 - USDC 流入 的价值差
  LP 承受巨大损失

StableSwap 池（A=100）:
  曲线在偏离平衡点后快速弯曲
  自动减少 USDC → USDT 的交易量
  保护 LP 的 USDT 不被过度消耗
  同时仍然允许合理的交易量
```

## 多币 StableSwap

StableSwap 不限于两种代币，可以支持多种稳定币：

```
3 币池: USDC + USDT + DAI
  更高的资金效率
  更多的交易路径
  但更复杂的数学（需要更高维度的 Newton 迭代）

公式（3 币）:
  A × (x + y + z)³ + 27 × x × y × z = D³/27 + A × D³
```

Sui 上的 StableSwap 实现通常支持 2-4 种代币的池，满足大多数稳定币互换需求。
