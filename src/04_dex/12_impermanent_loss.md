# 4.12 Impermanent Loss

无常损失（Impermanent Loss, IL）是 LP 面临的最大风险之一。理解 IL 的数学本质，才能做出明智的 LP 决策。

## 什么是无常损失

当 LP 将代币存入 AMM 池后，如果代币的相对价格发生变化，LP 的持仓价值会**低于简单持有同样代币的价值**。这个差额就是无常损失。

```
为什么叫"无常"：
  如果价格回到存入时的水平，IL 消失（损失 = 0）
  只有在价格偏离时才存在

但现实是：
  价格很少回到原来的水平
  所以 IL 通常是"永久的"
```

## IL 的数学推导

### 初始状态

```
LP 存入等值的 A 和 B:
  amount_a = D_A / P_A （D_A 美元价值的 A）
  amount_b = D_B / P_B （D_B 美元价值的 B）
  且 D_A = D_B（等值存入）

HODL 价值 = D_A + D_B = 2D
```

### 价格变化后

```
假设 A 相对 B 的价格变化了 p 倍（p > 1 表示 A 升值）

AMM 中的新储备量（k 不变）:
  x_new = sqrt(k / p)
  y_new = sqrt(k × p)

LP 持有池的一半份额:
  AMM 价值 = x_new × P_A × p + y_new × P_B

HODL 价值 = D × (1 + p) / 2（假设初始等值）

IL = AMM 价值 / HODL 价值 - 1
   = 2 × sqrt(p) / (1 + p) - 1
```

### IL 公式

```
IL = 2 × sqrt(price_ratio) / (1 + price_ratio) - 1

其中 price_ratio = 新价格 / 旧价格
```

## IL 数值表

| 价格变化 | price_ratio | IL | 说明 |
|---------|-------------|-----|------|
| 不变 | 1.00 | 0.00% | 无损失 |
| ±25% | 1.25 | -0.60% | 轻微 |
| ±50% | 1.50 | -2.02% | |
| 2x / 0.5x | 2.00 | -5.72% | 价格翻倍或腰斩 |
| 3x / 0.33x | 3.00 | -13.40% | |
| 5x / 0.2x | 5.00 | -25.46% | |
| 10x / 0.1x | 10.00 | -40.51% | |

### 关键洞察

```
1. 价格变化越大，IL 越大
2. IL 是对称的：涨 2x 和跌 50% 的 IL 一样
3. 即使价格只变化 2 倍，IL 已经达到 5.72%
4. 对于波动性大的代币，IL 可能远超手续费收入
```

## 实际案例

```
LP 在 SUI/USDC 池中存入：
  1000 SUI ($2,000) + 2000 USDC ($2,000) = $4,000 总价值

场景 A：SUI 涨到 $3
  HODL: 1000×$3 + $2000 = $5,000
  AMM:  池自动卖出 SUI，LP 持有更少 SUI、更多 USDC
        LP 价值 ≈ $4,885
  IL = ($4,885 - $5,000) / $5,000 = -2.30%

场景 B：SUI 跌到 $1
  HODL: 1000×$1 + $2000 = $3,000
  AMM:  池自动买入 SUI，LP 持有更多 SUI、更少 USDC
        LP 价值 ≈ $2,930
  IL = ($2,930 - $3,000) / $3,000 = -2.33%
```

## IL vs 手续费收入

LP 的实际收益 = 手续费收入 - 无常损失

```
判断 LP 是否盈利的关键:

手续费 APY > |IL| → LP 盈利
手续费 APY < |IL| → LP 亏损

例：
  SUI/USDC 池
  手续费 APY: 15%
  SUI 年波动率: ~100%
  预期 IL: ~5-10%（取决于实际价格路径）
  净收益: +5% ~ +10% → 大多数时候盈利

但如果 SUI 暴涨 5x:
  IL: -25.46%
  手续费收入: ~15%
  净收益: -10.46% → 亏损！
```

## 不同 DEX 类型的 IL

| DEX 类型 | IL 特征 | 说明 |
|---------|---------|------|
| CPMM | 标准IL | 全范围都有 IL |
| CLMM | 区间内 IL 更大 | 集中流动性放大了 IL |
| DLMM | 取决于 bin 分布 | 类似 CLMM |
| Orderbook | 无 IL | Maker 不承担 IL |

### CLMM 的 IL 特征

```
CLMM LP 在窄区间提供流动性:
  → 资本效率高（同等的资金提供更深流动性）
  → 但如果价格移出区间:
    → 所有流动性变成单一代币
    → IL 更大

例：SUI = $2.00，LP 在 $1.80-$2.20 区间提供流动性
  → SUI 涨到 $2.30 → LP 只有 USDC（所有 SUI 被换成 USDC）
  → IL 非常大（相当于价格变化被完全捕捉）
```

## IL 管理策略

```
1. 选择低波动性交易对
   稳定币对: IL ≈ 0
   主流币对: IL 中等
   山寨币对: IL 很大

2. 选择高交易量的池
   交易量 → 手续费收入 → 补偿 IL

3. 使用 CLMM 时选择宽区间
   宽区间 → IL 接近 CPMM
   窄区间 → IL 被放大

4. 定期再平衡
   价格移出区间后，调整区间到新价格周围

5. 短期 LP vs 长期 LP
   短期: IL 还没完全展开就撤出
   长期: IL 可能更大，但手续费累积也更多
```
