# 4.20 DLMM 概念

DLMM（Dynamic Liquidity Market Maker）是 CLMM 的进一步进化。它用离散的 Bin 替代连续的 Tick 区间，每个 Bin 是独立的价格点，LP 可精确选择在哪些价格点提供流动性。

## 流动性模型的三个阶段

```
阶段 1: CPMM — 全区间均匀
  ████████████████████████████████████████  (0, +inf)

阶段 2: CLMM — 选择性区间
  ░░░░░████████████████░░░░░░░░░░░░░░░░░  [tick_lower, tick_upper]

阶段 3: DLMM — 精确价格点
  ░░░░░░░░│░│███│█│░░░░░░░░░░░░░░░░░░░  每个 Bin 独立
```

## Bin vs Tick

```
Tick (CLMM):
  流动性在 [tick_lower, tick_upper] 之间连续分布
  LP 选择一个"区间"

Bin (DLMM):
  每个 Bin = 一个独立价格点
  LP 选择一个或多个"点"
  不同 Bin 的流动性可以不同

比喻: Tick = 房间的两扇门（定义范围）
      Bin  = 独立的格子（完全离散）
```

## 资本效率对比

```
场景: SUI/USDC, 当前 $2.00, 投入 $10,000

CPMM:  有效流动性在 $1.99-$2.01 ≈ 极少
CLMM:  区间 $1.90-$2.10, 有效 ≈ $1,000  (10x CPMM)
DLMM:  5 个 Bin, 50% 放在 $2.00, 有效 ≈ $5,000 (50x CPMM)

模型   | 效率       | 精度     | 灵活性    | 复杂度
───────|───────────|──────────|───────────|───────
CPMM   | 1x        | 连续     | 无        | 最低
CLMM   | 20-400x   | 区间连续 | 区间选择   | 中等
DLMM   | 50-1000x  | 离散精确 | Bin 级别   | 较高
```

## 流动性分布策略

```
1. 均匀分布:
   $1.96  $1.98  $2.00  $2.02  $2.04
    ███    ███    ███    ███    ███

2. 钟形分布 (推荐):
   $1.96  $1.98  $2.00  $2.02  $2.04
    █      ███    █████  ███    █
                     ↑ 最多

3. 单 Bin:
   $1.96  $1.98  $2.00  $2.02  $2.04
    ░      ░      █████  ░      ░
                    100%

4. 曲线分布:
   $1.96  $1.98  $2.00  $2.02  $2.04
    █      ██     █████  ██     █
```

## Sui 并行执行优势

```
DLMM 的 Bin 天然独立:
  Tx-1: Swap 涉及 Bin 3  ──▶  并行
  Tx-2: LP 存入 Bin 5    ──▶  并行
  Tx-3: Swap 涉及 Bin 4  ──▶  并行
  Tx-4: LP 从 Bin 2 取出 ──▶  并行

Sui 对象模型 + 并行 = DLMM 理想平台

对象存储方案:
  A: Bin 作为 Pool dynamic field → 简单但 Swap 串行
  B: Bin 作为独立共享对象 → 完全并行但对象多
  C: Bin Group → 折中方案
```

## 生态参考

```
Meteora (Solana): 最知名的 DLMM 协议
  - 每个 Bin 独立管理
  - Bin 步长可配置: 稳定币 0.01%, 主流 0.05-0.1%, 长尾 1-5%

Sui 生态: 目前以 CLMM 为主 (Cetus, Turbos)
  Sui 的并行执行 + 对象模型 → DLMM 在 Sui 上潜力巨大
```
