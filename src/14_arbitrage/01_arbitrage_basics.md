# 12.1 套利的本质与无套利原理

## 一价定律

同一个资产在两个市场上应该有相同的价格。如果有差异，就存在套利机会：在低价市场买入，在高价市场卖出。

$$P_A \neq P_B \Rightarrow \text{套利机会}$$

## 套利为什么"总是"存在

理论上套利机会应该瞬间消失。但在 DeFi 中，由于以下原因，套利持续存在：

| 原因 | 说明 |
|------|------|
| 区块时间 | 两个交易不可能在同一瞬间执行 |
| Gas 成本 | 小额价差可能被 Gas 吃掉 |
| 流动性不均 | 不同 DEX 的池子深度不同 |
| 延迟 | 链上状态更新需要时间 |
| 协议机制 | AMM 的非线性定价天然产生价差 |

## 套利的分类

### 按风险分

| 类型 | 风险 | 例子 |
|------|------|------|
| 纯套利（Pure Arbitrage） | 零风险，同一资产跨市场 | DEX A 买 SUI，DEX B 卖 SUI |
| 统计套利（Statistical Arbitrage） | 有风险，基于历史相关性 | 相关资产对的价格偏离均值 |
| 事件套利（Event Arbitrage） | 有风险，基于预期事件 | 预言机更新前的仓位布局 |

### 按协议组合分

| 类型 | 涉及协议 | 描述 |
|------|----------|------|
| DEX-DEX | 两个 DEX | 跨池价差 |
| DEX-借贷 | DEX + Lending | 清算套利 |
| 闪电贷-X | 闪电贷 + 任意协议 | 零资本套利 |
| CDP-DEX | CDP + DEX | 稳定币锚定套利 |
| 衍生品-现货 | Perp + DEX | 基差套利 |

## 无套利条件

当市场均衡时，以下条件成立：

$$P_{DEX_A} = P_{DEX_B} = P_{Oracle} = P_{Perp\_Mark}$$

任何偏离都意味着套利机会。套利者的行为会将价格推回均衡——这是 DeFi 价格发现的核心机制。

## 套利的经济学

```
套利利润 = 价差收益 - Gas 成本 - 滑点损失 - 时间风险
```

只有当利润 > 成本时，套利才会被执行。这意味着：
- 小额价差（< Gas）不会被套利
- 低流动性池的滑点可能吞没利润
- 高 Gas 时期只有大额套利才有利可图
