# 第 14 章 套利：价格修复者与 MEV 生态

## 套利是 DeFi 的"引力"

前面十章讲了 DEX、预言机、借贷、闪电贷、杠杆——这些协议各自运转，但它们之间有"力"在拉扯。套利就是这种力的具象化：**当不同协议对同一资产的定价不一致时，套利者通过交易消除价差。**

套利不是"作弊"。它是市场效率的引擎——没有套利者，DEX 的价格会长期偏离市场，借贷协议的清算会无人执行，稳定币会持续脱锚。

但套利也是一把双刃剑：三明治攻击、MEV 提取、Gas 竞价战争——这些都在"套利"的名下发生。

## 本章目标

- 理解套利如何让不同市场价格回归，也如何变成攻击路径。
- 掌握 DEX 价差、闪电贷、清算套利和跨协议组合套利。
- 区分健康套利、MEV 提取和用户伤害型三明治攻击。
- 能从 Gas、滑点、竞争和失败交易角度评估套利策略。

## 先修知识

- 理解 DEX 价格冲击、借贷清算和 PTB 原子性。
- 能阅读基础 TypeScript 策略示例。

## 本章小结

套利是 DeFi 的价格修复机制，但它不是无害背景进程。套利者、清算者和 MEV 搜索者都在利用公开状态和交易排序，协议必须接受这种现实并设计保护用户和系统的边界。
