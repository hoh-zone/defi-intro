# 11.8 Sui 做市策略实例

## Cetus CLMM 做市策略

Cetus 是 Sui 上最大的 CLMM DEX。以下是实际可行的做市策略：

### 策略 1：宽区间被动 LP

```
适用：不想频繁管理的用户
交易对：SUI/USDC
区间：当前价 ±30%
费率档位：0.25%
预期 APR：20-40%（手续费 + 激励）
管理频率：每周检查一次

操作步骤：
1. 在 Cetus 上选择 SUI/USDC 池
2. 设置区间为当前价 ±30%
3. 存入等值 SUI 和 USDC
4. 质押 LP NFT 到挖矿合约
5. 每周检查是否需要再平衡
```

### 策略 2：窄区间主动 LP

```
适用：愿意每天管理的用户
交易对：SUI/USDC
区间：当前价 ±5%
费率档位：0.05%
预期 APR：60-120%（手续费 + 激励）
管理频率：每天检查，价格穿出区间时再平衡

风险：
  每次再平衡产生滑点损失
  激励代币 CETUS 的价格波动
  SUI 单边行情导致频繁再平衡
```

## DeepBook 订单簿做市

DeepBook 是 Sui 上的 CLOB 订单簿。做市方式与 AMM 完全不同：

```
策略：双边报价做市
交易对：SUI/USDC
价差：0.1%-0.3%
订单量：每边 500-2000 USDC
更新频率：每分钟调整报价

核心逻辑：
  mid_price = (best_bid + best_ask) / 2
  my_bid = mid_price - spread/2
  my_ask = mid_price + spread/2

  如果库存偏斜：
    库存多 → 降低 bid（不急于买入），降低 ask（急于卖出）
    库存少 → 提高 bid（急于买入），提高 ask（不急于卖出）
```

### DeepBook 做市的 PTB 流程

```
PTB（Programmable Transaction Block）：
  1. 从 DeepBook Account 取出 USDC → 挂 bid 单
  2. 从 DeepBook Account 取出 SUI → 挂 ask 单
  3. 检查已有订单状态
  4. 取消过时订单
  5. 挂新报价

Sui 的 PTB 允许在单笔交易中完成所有步骤，
这是 Sui 相比 EVM 链做市的巨大优势。
```

## Navi 杠杆挖矿策略

### 策略：SUI 杠杆做多挖矿

```
初始资金：1000 SUI
目标杠杆：2.5x
协议：Navi（借贷）+ Cetus（DEX/LP）

步骤：
1. 存入 1000 SUI 到 Navi
2. 借出 800 USDC（80% LTV）
3. 在 Cetus 将 800 USDC 换成 ~800 SUI
4. 存入 800 SUI 到 Navi
5. 借出 640 USDC
6. 换成 SUI，再次存入...
7. 最终：~2500 SUI 存入，~1500 USDC 借出

收益计算：
  存款利息（SUI）：2500 × 2% = 50 SUI
  存款激励（NAVX）：2500 × 15% = 375 NAVX
  借款利息（USDC）：1500 × 5% = 75 USDC
  借款激励（NAVX）：1500 × 10% = 150 NAVX
  LP 收益（如果再配对做 LP）：~100 SUI

  总收益（假设 NAVX = $0.1）：
    50 + 37.5 + (-75) + 15 + 100 = 127.5 SUI 等值
    ROI = 127.5/1000 ≈ 12.75%
```

**警惕**：这个 12.75% 的 ROI 看起来不高，但承担了 2.5x 的清算风险。SUI 跌 20% 可能触发清算。

## 收益策略的风险等级表

```
策略               预期年化    风险等级    管理频率    最坏情况
───────────────────────────────────────────────────────────
LSD（质押）         3-8%       低          无          质押锁定期
宽区间 LP           20-40%     中          每周        30% IL
窄区间 LP           60-120%    中高        每天        区间穿出 + IL
订单簿做市          30-60%     中高        实时        库存偏斜
网格交易            40-80%     中          自动        突破区间
收益金库            15-30%     中          无          策略失败
杠杆挖矿 2x         40-80%     高          每天        清算
杠杆挖矿 3x+        60-150%    极高        实时        快速清算
Delta 中性          20-40%     中          每天        对冲失效
资金费率套利        10-25%     中低        每天        费率反转
```

## 策略选择决策树

```
你的风险承受能力？
├─ 低 → LSD 或 宽区间 LP
├─ 中 → 收益金库 或 网格交易
└─ 高 → 你有多少时间管理？
    ├─ 每天 → 窄区间 LP 或 订单簿做市
    └─ 实时 → 杠杆挖矿 或 Delta 中性
```

## 全章风险总结

| 风险类别 | 影响策略 | 共性 |
|---|---|---|
| 无常损失 | CLMM LP, 网格 | 价格变化导致资产价值低于持有 |
| 库存偏斜 | 订单簿 MM | 单边行情导致持仓失衡 |
| 清算 | 杠杆挖矿 | 价格反向波动触发强制平仓 |
| 策略失效 | Vault, Delta 中性 | 市场条件变化导致策略不再盈利 |
| 协议风险 | 所有策略 | 底层协议被攻击或出故障 |
| 激励衰减 | LP, 杠杆挖矿 | 奖励减少导致收益下降 |

主动收益策略的核心教训：**任何看起来"稳赚"的策略，都在某个你没想到的地方隐藏着风险**。理解这些风险的唯一方式是读懂代码、算清数学、做好压力测试。
