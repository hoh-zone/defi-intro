# 第 11 章 自动做市与收益策略

## 从"持有"到"经营"

前一章讲的 LSD 是被动收益：质押→等待→收获。本章进入主动收益领域——你需要做出决策、承担风险、持续管理仓位。

做市（Market Making）是最古老的金融业务之一。传统做市商通过持续提供买卖报价赚取价差。DeFi 中，任何人都可以成为做市商——向 AMM 池或订单簿提供流动性，赚取交易手续费。

但"任何人都行"不意味着"任何人都能赚"。做市的收益来自手续费和激励，成本来自无常损失（Impermanent Loss）和资金占用。净收益可能是正的，也可能是负的。

## 警惕视角

主动收益策略的核心风险是**你以为你在做市，其实你在赌方向**。如果你的 LP 仓位有明显的方向性暴露，那不是做市，是杠杆持仓——只是包装得像做市而已。
