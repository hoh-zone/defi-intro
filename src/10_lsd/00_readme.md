# 第 10 章 LSD 与收益层

## 质押收益如何变成可流通资产

LSD（Liquid Staking Derivative）解决一个基本矛盾：质押的资产被锁定无法使用，但 DeFi 需要这些资产参与流动性。

LST（Liquid Staking Token）是解决方案：它代表你质押的资产 + 累积的收益，并且可以在 DeFi 中自由流通。
